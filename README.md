# Social Machine

AI-powered social media marketing pipeline for Claude Code.

Social Machine turns your project into a content machine. It scans your codebase for brand identity, researches current trends, generates post ideas with multiple caption variants, creates publish-ready graphics (including real meme templates), and posts to your social platforms — all from a single `/social` command.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/social-machine-skill.git
cd social-machine-skill

# Install the skill (creates symlinks to ~/.claude/skills/)
chmod +x scripts/install.sh
./scripts/install.sh

# Navigate to any project and run:
/social
```

That's it. Social Machine will walk you through setup — scanning your brand, choosing your tools, and creating your first post.

## Features

### Content Creation
- **Brand-aware**: Auto-extracts colors, logos, features, and tone from your codebase
- **Trend-informed**: Researches current social media trends before creating content
- **8 content types**: Feature spotlight, quote card, stat graphic, before/after, tip/how-to, announcement, testimonial, and meme
- **3 caption variants**: Question hook, stat hook, and story hook for every post
- **Content variety tracking**: Tracks post history and prevents repetitive content

### Meme Engine
- **Real meme templates**: Drake, Expanding Brain, Distracted Boyfriend, Change My Mind, and 200+ more via memegen.link API
- **Custom meme layouts**: Alignment charts, tweet screenshots, hot takes, starter packs, "this or that" — all via HTML/CSS
- **Trend scouting**: Searches for what's currently viral and evaluates brand fit
- **Style transfer**: Takes trending formats and adapts them to your brand voice
- **Auto-watermarking**: Stamps your logo + @handle on every meme

### Graphics
- **Multi-platform**: Optimized dimensions for Instagram, X, TikTok, LinkedIn, Facebook
- **Stock photos**: Finds royalty-free images from Unsplash, Pexels, Pixabay with license tracking
- **Swappable providers**: HTML/CSS screenshot (default), memegen API, Figma MCP, or SVG

### Publishing
- **Manual mode** (default): Saves graphics + captions for you to post
- **API mode**: Posts directly via Instagram Graph API, X API v2, TikTok API
- **Browser mode**: Automates posting via browser UI

## Architecture

```
/social (orchestrator)
  ├── /social-scan       → Brand extraction from codebase
  ├── /social-research   → Trend research via web search
  ├── /social-ideate     → Content ideas + captions
  ├── /social-meme       → Meme trend scouting + generation
  ├── /social-stock      → Royalty-free stock images
  ├── /social-design     → Graphics creation (routes to provider)
  │   ├── html-screenshot  → HTML/CSS → headless browser screenshot
  │   ├── memegen-api      → Real meme templates + logo watermark
  │   ├── meme-layouts     → Custom meme CSS templates
  │   ├── figma            → Figma MCP integration
  │   └── svg              → Pure SVG fallback
  ├── /social-capture    → App screenshots
  └── /social-post       → Publishing (routes to provider)
      ├── manual           → Save files for manual posting
      ├── api              → Platform API posting
      └── browser          → Browser automation posting
```

Each stage works independently or chains together via the orchestrator.

## Per-Project State

Social Machine stores all project-specific data in `.social-machine/`:

```
.social-machine/
├── brand.md            # Brand identity (auto-generated from codebase)
├── config.md           # Provider and platform settings
├── credentials.env     # API keys (auto-gitignored)
├── research-latest.md  # Latest trend research
├── history/
│   └── posts.md        # Post history for variety tracking
└── output/             # Generated graphics, captions, stock photos
    ├── stock/           # Downloaded stock images + LICENSES.md
    └── captions/        # Platform-specific caption files
```

## Requirements

- [Claude Code](https://claude.com/claude-code) with skill support
- For HTML/CSS graphics: Playwright (`npx playwright`) or a headless browser
- For meme templates: Internet connection (memegen.link API, free, no auth)
- For watermarking: Python 3 (auto-creates venv with Pillow)
- For Figma graphics: Figma MCP server configured
- For API posting: Platform API keys (guided setup via `/social-config`)
